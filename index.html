Guia de Estudo para a AWS SAA-C03

Bem-vindo ao seu guia de estudos prático para a certificação AWS Certified Solutions Architect - Associate (SAA-C03). Este material foi desenhado para ser um manual de revisão rápida, focado nos 81 serviços distribuídos em 16 categorias essenciais do escopo do exame.

Selecione uma categoria abaixo para iniciar sua jornada de aprendizado.

  • Compute (Computação)
  • Storage (Armazenamento)
  • Database (Banco de Dados)
  • Networking & Content Delivery
  • Containers (Contêineres)
  • Serverless (Sem Servidor)
  • Migration & Transfer
  • Application Integration
  • Analytics (Análise de Dados)
  • Machine Learning
  • Front-end Web & Mobile
  • Media Services
  • Security, Identity, & Compliance
  • Management & Governance
  • AWS Cost Management
  • Developer Tools
